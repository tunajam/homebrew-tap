# Tunajam Homebrew Tap

Homebrew formulae for [Tunajam](https://tunajam.com) tools.

## Installation

```bash
brew tap tunajam/tap
brew install packs
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [packs](https://packs.sh) | Agent skill package manager - share and install AI agent skills |

## Updating

```bash
brew update
brew upgrade packs
```
